# AFO

[English](README_EN.md) | Русский

**Ambient Flow Orchestrator** — приложение для Windows, которое следит за тем, чем ты занят, и подстраивает рабочее окружение под текущую задачу.

---

## Зачем это нужно

Когда работаешь за компьютером, хочется чтобы всё было "под рукой": нужный фоновый звук, комфортная яркость экрана, минимум отвлекающих уведомлений. AFO делает это автоматически — смотрит какие приложения открыты, определяет режим (работа, учёба, отдых) и настраивает окружение.

## Что умеет

- Следит за активными приложениями и временем работы в них
- Понимает когда ты в "потоке", а когда просто листаешь ленту
- Включает фоновые звуки (дождь, кафе, шум) когда надо сосредоточиться
- Переключает экран в тёплый режим вечером
- Напоминает сделать перерыв
- Автозапуск при старте Windows (настраивается в приложении)

## Установка

Нужен Windows 10/11 и Python 3.10+

```
pip install -r requirements.txt
```

## Сборка exe

Запусти `build.bat` — он соберёт `dist/AFO.exe`

## Автозапуск

В настройках приложения есть переключатель "Автозапуск" — он добавит AFO в автозагрузку Windows.

> Работает только в exe версии

## API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | /api/status | текущее состояние |
| GET | /api/stats | статистика за день |
| GET/POST | /api/config | настройки |
| POST | /api/sound | управление звуком |
| POST | /api/break | запустить перерыв |
| GET/POST | /api/autostart | автозагрузка |
